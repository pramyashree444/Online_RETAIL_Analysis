# Online_RETAIL_Analysis
RETAIL_SALES_ANALYSIS

## Objective
Analyze online retail sales data to identify revenue trends, top products, and top-performing countries using Python and SQL.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- Google Colab

## Dashboards
- Monthly Revenue Trend
- Top 10 Products by Revenue
- Top Countries by Revenue

## Key Insights
- Revenue shows seasonal patterns across months
- A small number of products contribute to major revenue
- UK and European countries dominate sales

## Files
- dashboards/: contains exported dashboard images
- notebook/: analysis notebook
